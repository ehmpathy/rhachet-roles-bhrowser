# 🦎 rhachet-roles-bhrowser

browser automation roles and skills, via [rhachet](https://github.com/ehmpathy/rhachet)

## purpose

bhrowser = browser + automation

this repo provides roles for browser automation — playbooks that drive browsers, visual assessment that catches regressions, all while blended in seamlessly.

## install

```sh
npm install rhachet-roles-bhrowser
```

## roles

### 📽️ playwright

- **scale**: workflow-level, browser automation
- **focus**: compose repeatable playbooks — clicks, fills, navigates, workflows
- **maximizes**: reliable, maintainable browser automation scripts

used to craft automation playbooks that drive browsers through complex workflows.

### 📐 inspector

- **scale**: visual-level, design assessment
- **focus**: inspect performance, precision, and aesthetics — screenshots, diffs, regressions
- **maximizes**: visual accuracy and design coherence

used to assess browser renders, spot visual regressions, and ensure design consistency.

## mascot

this repo houses the chameleon 🦎 — who blends into any environment, watches with independent eyes, and strikes with precision.

why chameleons? because they embody the principles of good browser automation:

1. 🦎 blend in — adapt to any site, avoid detection
2. 👁️ independent eyes — watch DOM, network, and console simultaneously
3. 🎨 color shift — adapt behavior per context
4. 👅 precise tongue — grab exactly the element you need
5. 🐢 deliberate movement — careful, measured actions

**vibe phrases:**
- "blend in..."
- "adapt..."
- "watch closely..."
- "got it 🦎"

**the chameleon wields:**
- 📽️ projector — for playwright — to project recorded performances onto browsers
- 📐 set square — for inspector — to inspect performance, precision, and aesthetics

## why open source

browser automation should be accessible. we all benefit from shared patterns for test, scrape, and workflow automation.

**for developers**: contribute upstream to improve patterns, fork to specialize for your use case.

**for teams**: leverage battle-tested automation roles instead of the wheel reinvented.

let's raise the floor. mutually.
