# blackbox criteria: inject prior browser skills

## episode experience

### usecase.1 = mechanic enrolls playwright role and learns browser patterns

```
given('mechanic needs browser automation for a new site')
  when('enrolls repo=bhrowser/role=playwright')
    then('briefs on stealth, auth, cache, diagnosis are available')
      sothat('mechanic understands patterns before code')
    then('skills browser.* are available')
      sothat('mechanic can drive browser from shell')
```

### usecase.2 = mechanic launches and manages browser session

```
given('mechanic has enrolled playwright role')
  when('runs browser.start')
    then('chromium launches with stealth configured')
    then('ws-endpoint file is written to .cache/')
      sothat('subsequent skills can connect')

given('browser is active')
  when('runs browser.describe')
    then('lists all tabs with index, url, title')
      sothat('mechanic can select correct tab')

given('browser is active')
  when('runs browser.stop')
    then('browser terminates gracefully')
    then('endpoint file is cleaned up')
```

### usecase.3 = mechanic runs automation via playbooks

```
given('browser is active')
  when('runs browser.action --play path/to/playbook.play.ts')
    then('playbook executes against specified tab')
    then('result is logged with url before/after')
      sothat('mechanic sees what happened')

given('playbook file does not exist')
  when('runs browser.action --play nonexistent.play.ts')
    then('error indicates playbook not found')
    then('exits with non-zero code')
```

### usecase.4 = mechanic captures diagnostic snapshots

```
given('browser is active with page loaded')
  when('runs browser.snapshot --focused')
    then('captures screenshot, html, console, network, storage, meta')
    then('files written to .cache/browser.$session/snapshot.$time/')
      sothat('mechanic can diagnose issues post-mortem')

given('browser is active')
  when('runs browser.snapshot --tab N --url "expected/path"')
    then('verifies tab N has expected URL')
    then('captures snapshot of that tab')

given('tab URL does not match expected')
  when('runs browser.snapshot --tab N --url "wrong/path"')
    then('error indicates URL mismatch')
    then('suggests browser.describe to find correct tab')
```

## exchange experience

### browser.start

```
given('no browser active for session')
  when('browser.start --session <name>')
    then('launches chromium with stealth')
    then('writes .cache/browser.$session/ws-endpoint')
    then('outputs success with endpoint path')

given('browser already active for session')
  when('browser.start --session <name>')
    then('outputs that browser is already active')
    then('shows extant endpoint path')

given('--mode HEADLESS')
  when('browser.start --mode HEADLESS')
    then('launches in headless mode')

given('--mode HEADFUL')
  when('browser.start --mode HEADFUL')
    then('launches in headful mode for human interaction')
```

### browser.stop

```
given('browser active for session')
  when('browser.stop --session <name>')
    then('terminates browser process')
    then('removes ws-endpoint file')
    then('outputs success')

given('no browser active for session')
  when('browser.stop --session <name>')
    then('outputs that no browser is active')
```

### browser.describe

```
given('browser active with tabs')
  when('browser.describe --session <name>')
    then('lists each tab with index, url, title')
    then('indicates which tab is focused')

given('no browser active')
  when('browser.describe')
    then('error indicates no browser found')
```

### browser.action

```
given('browser active, playbook exists')
  when('browser.action --play <path>')
    then('connects to browser via CDP')
    then('executes action() export from playbook')
    then('logs url before, url after, result')

given('--tab N specified')
  when('browser.action --play <path> --tab N')
    then('executes on tab N instead of tab 0')

given('tab index out of range')
  when('browser.action --play <path> --tab 99')
    then('error indicates tab not found')
```

### browser.snapshot

```
given('browser active, --focused')
  when('browser.snapshot --focused')
    then('finds focused tab automatically')
    then('captures all artifacts to snapshot directory')

given('browser active, --tab N --url "x"')
  when('browser.snapshot --tab N --url "x"')
    then('verifies tab N contains url "x"')
    then('captures all artifacts')

given('--tab without --url')
  when('browser.snapshot --tab N')
    then('error requires --url with --tab')
    then('suggests --focused or browser.describe')

given('subcommand specified')
  when('browser.snapshot screen --focused')
    then('captures only screenshot (not all artifacts)')
```
